# Claude Study Note Rules

이 저장소는 질문과 답변을 쌓는 개인 학습 노트 사이트입니다.

## 사용자가 "노트추가"라고 말하면

방금 답변한 내용을 초보자용 학습 노트로 바꿔서 저장합니다.

1. `경제`, `코딩`, `AI`, `생활` 중 가장 맞는 카테고리를 고릅니다.
2. 세부 분류는 태그로 넣습니다.
3. `docs/templates/note-template.md` 형식을 따릅니다.
4. 새 노트를 `docs/<category>/` 아래에 만듭니다.
5. 해당 카테고리 `index.md`에 링크를 추가합니다.
6. `docs/review/anki-cards.md`와 `docs/review/anki-cards.csv`에 복습 카드를 추가합니다.
7. 필요하면 `mkdocs.yml`의 nav에도 추가합니다.
8. `mkdocs build --strict`로 검증합니다.
9. 사용자가 요청하면 GitHub에 커밋/푸시합니다.

## 글쓰기 톤

- 재무제표나 코딩을 모르는 사람도 이해할 수 있게 씁니다.
- 어려운 말은 쉬운 예시로 풉니다.
- 노트 하나는 다시 읽기 쉬워야 합니다.
- 암기할 것은 복습 카드로 분리합니다.

## 카테고리 기준

- 경제: 주식, 재무제표, 거시경제, 투자, 트레이딩
- 코딩: 프로그래밍, 오류, 자동화, GitHub, 웹 개발
- AI: Codex, Claude, 프롬프트, AI 도구와 업무흐름
- 생활: 업무 습관, 의사결정, 일상 지식, 기타 배움
