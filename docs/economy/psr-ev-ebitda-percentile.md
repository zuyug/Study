# PSR/EV/EBITDA 분위수 쉽게 이해하기

<div class="note-meta" markdown>
**카테고리:** 경제  
**난이도:** 초보자  
**핵심 질문:** PSR/EV/EBITDA 분위수는 무엇을 보여주는가?
</div>

!!! summary "한 줄 요약"
    **PSR/EV/EBITDA 분위수는 이 종목이 매출이나 현금창출력 대비 지금 싼 구간인지, 평균인지, 비싼 구간인지 위치로 보여주는 지표**입니다.

## 먼저 용어 나누기

| 용어 | 쉽게 말하면 | 보는 것 |
| --- | --- | --- |
| PSR | 매출 기준 가격표 | 매출에 비해 시가총액이 몇 배인가 |
| EV/EBITDA | 현금창출력 기준 가격표 | 기업가치가 본업의 현금창출력에 비해 몇 배인가 |
| 분위수 | 현재 위치 표시 | 과거나 같은 업종 안에서 어느 정도로 높은가 |

## PSR: 매출에 비해 비싼가?

PSR은 회사의 **매출액**에 비해 시장에서 회사 전체를 얼마로 평가하는지 보는 지표입니다.

```text
PSR = 시가총액 / 매출액
```

예를 들어 회사의 시가총액이 1조 원이고 연매출이 5,000억 원이면:

```text
PSR = 1조 / 5,000억 = 2배
```

매출 1원당 시장이 2원어치 가격을 붙이고 있다는 뜻으로 볼 수 있습니다.

!!! warning "주의할 점"
    PSR이 낮다고 무조건 좋은 종목은 아닙니다. 매출은 크지만 이익이 거의 없거나 적자가 계속되는 회사도 있을 수 있습니다.

## EV/EBITDA: 현금창출력에 비해 비싼가?

EV/EBITDA는 기업가치를 본업의 현금창출력과 비교하는 지표입니다.

```text
EV/EBITDA = 기업가치(EV) / EBITDA
```

- **EV:** 시가총액에 순차입금 등을 반영한 기업가치
- **EBITDA:** 이자, 세금, 감가상각비를 빼기 전 이익

쉽게 말하면 “회사를 통째로 사는 가격이 본업으로 벌어들이는 돈의 몇 배인가”를 보는 방식입니다.

!!! warning "주의할 점"
    EV/EBITDA가 낮아도 업황이 나빠지고 있거나 부채 부담이 크면 저평가가 아니라 위험 신호일 수 있습니다.

## 분위수: 지금 위치가 높은가 낮은가?

분위수는 현재 지표가 **과거 데이터나 같은 업종 종목들 안에서 어느 위치인지** 보여줍니다.

예를 들어 어떤 종목의 PSR이 과거 5년 기준 **90분위**라면:

- 과거 값 100개 중 약 90개보다 현재 PSR이 높다
- 과거 기준으로 꽤 비싼 구간일 가능성이 있다

반대로 **20분위**라면:

- 과거 값 100개 중 약 20개보다만 현재 PSR이 높다
- 과거 기준으로 낮은 구간일 가능성이 있다

## 단타 프로그램에서는 어떻게 쓰나?

PSR/EV/EBITDA 분위수는 1분봉 매수 신호처럼 바로 쓰기보다는 **종목 필터와 리스크 판단**에 더 적합합니다.

| 상황 | 해석 |
| --- | --- |
| PSR 95분위 + 뉴스 과열 + 거래량 급증 | 단기 급등은 가능하지만 고평가 부담이 큼 |
| EV/EBITDA 20분위 + 수급 개선 + 기술적 돌파 | 밸류 부담이 낮은 반등 후보로 볼 수 있음 |
| 분위수 높음 + 실적 기대 둔화 | 기대가 꺾이면 주가 변동성이 커질 수 있음 |

## 흔한 오해

- 분위수가 낮다고 무조건 매수 신호는 아닙니다.
- 분위수가 높다고 무조건 매도 신호도 아닙니다.
- 성장주와 경기민감주는 적정 PSR이나 EV/EBITDA 수준이 다를 수 있습니다.
- 같은 업종 비교와 과거 자기 자신 비교를 구분해야 합니다.

## 다시 기억할 문장

!!! quote
    PSR/EV/EBITDA 분위수는 현재 주가가 매출과 현금창출력 대비 어느 정도로 비싼 위치인지 보여주는 배경 점수입니다.
