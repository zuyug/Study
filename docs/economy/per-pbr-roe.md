# PER/PBR/ROE 쉽게 이해하기

<div class="note-meta" markdown>
**카테고리:** 경제  
**난이도:** 초보자  
**핵심 질문:** PER, PBR, ROE는 무엇인가?
</div>

!!! summary "한 줄 요약"
    **PER/PBR은 가격표**, **ROE는 실력표**입니다. PER과 PBR은 주가가 싼지 비싼지 볼 때 쓰고, ROE는 회사가 돈을 잘 버는지 볼 때 씁니다.

## 세 지표를 먼저 비교하기

| 지표 | 쉽게 말하면 | 보는 것 |
| --- | --- | --- |
| PER | 이익 기준 가격표 | 회사가 버는 돈에 비해 주가가 몇 배인가 |
| PBR | 자산 기준 가격표 | 회사가 가진 순자산에 비해 주가가 몇 배인가 |
| ROE | 돈 버는 실력 | 주주의 돈으로 이익을 얼마나 냈나 |

## PER: 이익에 비해 주가가 비싼가?

PER은 회사가 1년에 벌어들이는 **순이익**에 비해 시장에서 회사 전체를 얼마로 평가하는지 보는 지표입니다.

```text
PER = 시가총액 / 순이익
또는
PER = 주가 / 주당순이익(EPS)
```

예를 들어 회사 전체 가격이 1,000억 원이고 1년에 순이익을 100억 원 벌면:

```text
PER = 1,000억 / 100억 = 10배
```

이 말은 아주 단순히 보면 “지금 이익 수준이 계속된다는 가정에서 10년치 이익을 미리 주고 사는 셈”입니다.

!!! warning "주의할 점"
    PER이 낮다고 무조건 싼 주식은 아닙니다. 회사의 이익이 앞으로 줄어들 수 있다면, 낮은 PER은 기회가 아니라 경고일 수 있습니다.

## PBR: 자산에 비해 주가가 비싼가?

PBR은 회사가 가진 **순자산**에 비해 시장에서 회사 전체를 얼마로 평가하는지 보는 지표입니다.

순자산은 쉽게 말해:

```text
회사가 가진 것 - 갚아야 할 것
```

입니다.

```text
PBR = 시가총액 / 순자산
또는
PBR = 주가 / 주당순자산(BPS)
```

예를 들어 순자산이 1,000억 원인데 시가총액이 800억 원이면:

```text
PBR = 800억 / 1,000억 = 0.8배
```

장부상 순자산보다 낮은 가격에 거래된다고 볼 수 있습니다.

!!! warning "주의할 점"
    PBR이 1보다 낮아도 실제 자산의 질이 나쁘거나 미래 이익이 약하면 좋은 투자라고 보기 어렵습니다.

## ROE: 자기 돈으로 얼마나 잘 벌었나?

ROE는 회사가 **자기자본**을 활용해 얼마나 효율적으로 이익을 냈는지 보여줍니다.

```text
ROE = 순이익 / 자기자본
```

자기자본이 1,000억 원이고 1년에 순이익을 100억 원 냈다면:

```text
ROE = 100억 / 1,000억 = 10%
```

회사가 자기 돈 100원으로 1년에 10원을 벌어낸 셈입니다.

!!! warning "주의할 점"
    ROE가 높다고 무조건 좋은 회사는 아닙니다. 부채를 많이 써서 ROE가 높아진 경우도 있으므로 부채비율도 같이 보는 편이 안전합니다.

## 세 지표를 같이 보는 법

| 조합 | 초보자 해석 | 확인할 점 |
| --- | --- | --- |
| PER 낮음 + PBR 낮음 + ROE 높음 | 가격은 싸 보이는데 돈도 잘 버는 회사일 가능성 | 이익이 일시적인지, 업황이 꺾이는 중인지 |
| PER 낮음 + PBR 낮음 + ROE 낮음 | 싸 보이지만 시장 기대가 낮은 회사일 가능성 | 싼 주식이 아니라 약한 회사일 수 있음 |
| PER 높음 + PBR 높음 + ROE 높음 | 비싸지만 좋은 성장 회사로 평가받는 경우 | 성장이 둔화되면 주가가 크게 흔들릴 수 있음 |

## 다시 기억할 문장

!!! quote
    PER은 이익 대비 가격, PBR은 자산 대비 가격, ROE는 돈 버는 실력입니다.
