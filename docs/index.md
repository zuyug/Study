# Study Notes

질문하고 이해한 내용을 다시 읽기 쉽게 쌓아가는 개인 학습 노트입니다.

!!! tip "사용 방식"
    답변을 받은 뒤 **노트추가**라고 말하면 Codex나 Claude가 내용을 초보자용 노트로 정리하고, 카테고리와 태그를 붙이고, 복습 카드까지 갱신합니다.

## 카테고리

| 카테고리 | 용도 |
| --- | --- |
| [경제](economy/index.md) | 주식, 재무제표, 거시경제, 투자 지표 |
| [코딩](coding/index.md) | 프로그래밍, 오류 해결, 자동화, GitHub |
| [AI](ai/index.md) | Codex, Claude, 프롬프트, AI 도구 사용법 |
| [생활](life/index.md) | 일상 지식, 업무 습관, 의사결정, 기타 배움 |
| [복습](review/index.md) | Anki 카드, 다시 볼 핵심 질문 |

## 빠르게 찾는 법

- 상단 검색창에서 단어를 검색합니다. 예: `PER`, `재무제표`, `Python`
- 태그 화면에서 주제별로 모아봅니다.
- 노트 안의 **다시 기억할 문장**과 **복습 카드**를 먼저 보면 빠르게 복습할 수 있습니다.
