주식

PER/PBR/ROE 쉽게 이해하기

주식 초보자용 가치평가
한 줄 요약: PER과 PBR은 “지금 주가가 싼지 비싼지”를 보는 가격표이고, ROE는 “회사가 자기 돈으로 얼마나 잘 버는지”를 보는 실력표입니다.
PER 이익 기준 가격표. 회사가 버는 돈에 비해 주가가 몇 배인지 봅니다.
PBR 자산 기준 가격표. 회사의 순자산에 비해 주가가 몇 배인지 봅니다.
ROE 돈 버는 능력. 주주의 돈으로 이익을 얼마나 냈는지 봅니다.

PER: 이익에 비해 주가가 비싼가?

PER은 회사가 1년에 벌어들이는 순이익에 비해 시장에서 회사 전체를 얼마로 평가하는지 보는 지표입니다. 쉽게 말하면 “이 회사의 1년치 이익을 몇 년치 가격으로 사고 있나?”에 가깝습니다.

PER = 시가총액 / 순이익 또는 PER = 주가 / 주당순이익(EPS)

예를 들어 회사 전체 가격이 1,000억 원이고 1년에 순이익을 100억 원 벌면 PER은 10배입니다. 지금 이익 수준이 계속된다고 단순 가정하면 10년치 이익을 미리 주고 사는 셈입니다.

PBR: 자산에 비해 주가가 비싼가?

PBR은 회사가 가진 순자산에 비해 시장에서 회사 전체를 얼마로 평가하는지 보는 지표입니다. 순자산은 아주 쉽게 말해 “회사가 가진 것에서 갚아야 할 것을 뺀 값”입니다.

PBR = 시가총액 / 순자산 또는 PBR = 주가 / 주당순자산(BPS)

예를 들어 순자산이 1,000억 원인데 시가총액이 800억 원이면 PBR은 0.8배입니다. 장부상 순자산보다 낮은 가격에 거래된다고 볼 수 있지만, 자산의 질이나 미래 이익이 나쁘면 단순히 싸다고만 볼 수는 없습니다.

ROE: 자기 돈으로 얼마나 잘 벌었나?

ROE는 회사가 자기자본을 활용해 얼마나 효율적으로 이익을 냈는지 보여줍니다. 여기서 자기자본은 주주들의 몫, 즉 회사의 순자산과 가까운 개념으로 보면 됩니다.

ROE = 순이익 / 자기자본

자기자본이 1,000억 원이고 1년에 순이익을 100억 원 냈다면 ROE는 10%입니다. 회사가 자기 돈 100원으로 1년에 10원을 벌어낸 셈입니다.

세 지표를 같이 보는 법

조합 초보자 해석 주의할 점
PER 낮음 + PBR 낮음 + ROE 높음 가격은 싸 보이는데 돈도 잘 버는 회사일 가능성 일시적 이익인지, 업황이 꺾이는 중인지 확인 필요
PER 낮음 + PBR 낮음 + ROE 낮음 싸 보이지만 시장 기대가 낮은 회사일 가능성 싼 주식이 아니라 약한 회사일 수 있음
PER 높음 + PBR 높음 + ROE 높음 비싸지만 좋은 성장 회사로 평가받는 경우 성장이 둔화되면 주가가 크게 흔들릴 수 있음
초보자용 암기 문장

PER은 “이익 대비 가격”, PBR은 “자산 대비 가격”, ROE는 “돈 버는 실력”입니다. 그래서 PER/PBR은 얼마에 살지 판단할 때, ROE는 어떤 회사인지 판단할 때 먼저 봅니다.

무조건 낮거나 높으면 좋은가?

아닙니다. PER과 PBR이 낮아도 실적이 나빠지고 있으면 위험할 수 있고, ROE가 높아도 부채를 많이 써서 높아진 것일 수 있습니다. 지표는 정답지가 아니라 질문을 잘하게 해주는 도구입니다.
